# Serviak-FRASS
This codebase is the source code repository for the FRASS system. This codebase is only to be used by Dr. Bill and the related capstone teams affiliated with him.

## Project Summary

The Forest Resource Analysis System Software (FRASS) is an internet-accessible Software-as-a-Service (SaaS) program identifying financially optimal timber harvest rotation timing and asset value. This Highest and Best Use (HBU) solution gives landowners and resource managers an Income Capitalization Approach asset value. Here we detail how to realize this value with harvest timing for each timber stand; now and into perpetuity. This software solution presents a reliable and efficient system for scheduling economically optimal forest management activities. We do this while determining the value of timberland properties â€“ one timber stand at a time. In FRASS, tree growth data is combined with detailed information about the parcel. This engages delivered log market data predictions to give financially optimal timber harvest rotation dates and values into the future.

For more about FRASS visit [http://forest-econometrics.com/forest-econometrics-2/frass/](http://forest-econometrics.com/forest-econometrics-2/frass/)
